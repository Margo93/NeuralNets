========================================================================
    КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ
========================================================================
Предназначено для исследования архитектур нейронных сетей обратного распространения (обучение: обратное распространение ошибки) , позволяет 
создавать многослойные НС
удалять/добавлять слои НС
удалять/добавлять нейроны в существующие слои НС
задавать/изменять для каждого добавляемого нейрона 
	активационную функцию (сигмоид,тангенс,линейная функция : см. класс ActFuncs)
	число входов/выходов
	связи с нейронами предыдущего и следующего слоев
задавать для каждого добавляемого слоя
	скорость обучения
	число нейронов
во время обучения можно обучать
	всю сеть
	отдельные слои сети
	отдельные нейроны слоя
при этом можно отслеживать
	среднеквадратичную ошибку сети
	ошибку при тестировании
Входные данные представляют собой бинарные векторы произвольного размера, загружаемые из внешних файлов(см. класс inputs_loader)
Длина входных векторов устанавливается по минимальной длине среди всех загруженных векторов
После загрузки входных векторов из них составляются обучающая выборка и выборка для тестирования (класс LearningPair)
